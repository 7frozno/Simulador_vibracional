📊 Simulador de Análisis Vibracional

Un simulador profesional para análisis de vibraciones en bombas y equipos rotativos

🎯 Características

Análisis Espectral

  • FFT (Fast Fourier Transform): Análisis completo del espectro de frecuencias
  • Espectrograma: Visualización tiempo-frecuencia
  • Densidad Espectral de Potencia (PSD): Análisis energético

Análisis Temporal

  • RMS: Valor eficaz de la vibración
  • Pico y Pico-Pico: Amplitudes máximas
  • Factor de Cresta: Indicador de impactos
  • Kurtosis: Detecta transitorios e impactos
  • Asimetría: Características de distribución

🔧 Tipos de Falla Simulables

⚖️ Desbalance

  • Distribución no uniforme de masa
  • Dominancia en 1X
  • Bajo contenido de 2X

🔄 Desalineamiento

  • Desaligne axial/angular
  • Amplificación de armónicos pares
  • Bandas laterales significativas

⚡ Falla de Rodamiento

  • Defectos en pistas
  • Impulsos de alta frecuencia
  • Kurtosis muy elevada

💧 Cavitación

  • Formación de burbujas
  • Ruido de banda ancha
  • Deterioro rápido

✅ Equipo Sano

  • Funcionamiento normal
  • Bajos niveles
  • Espectro limpio

📊 Uso

Interfaz Gráfica (Streamlit)

Ejecutar la aplicación interactiva:

streamlit run app.py

La aplicación se abrirá en http://localhost:8501

Uso Programático

from src.signal_generator import SignalGenerator, SignalConfig
from src.signal_analyzer import SignalAnalyzer
from src.fault_simulator import FaultSimulator, FaultType
from src.diagnostics import DiagnosticEngine

# Generar señal
config = SignalConfig(sampling_rate=10000, duration=1.0, rpm=1500)
gen = SignalGenerator(config)
time, signal = gen.generate_healthy()

# Analizar
analyzer = SignalAnalyzer(10000)
rms = analyzer.calculate_rms(signal)
kurtosis = analyzer.calculate_kurtosis(signal)

# Diagnosticar
engine = DiagnosticEngine(10000, 1500)
diagnosis = engine.diagnose(signal)
